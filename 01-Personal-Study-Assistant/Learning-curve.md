Learning Path for Building AI Applications
1. Python Fundamentals (1-2 weeks)

Basic syntax, variables, data types
Functions and control flow
Working with libraries (pip, imports)
File handling and JSON

2. Essential Libraries (1 week)

requests: For making API calls
python-dotenv: For managing API keys safely
streamlit or gradio: For building simple UIs

3. Understanding LLMs & APIs (1 week)

What are Large Language Models (ChatGPT, Claude, etc.)
How to make API calls
Understanding prompts and responses
Token limits and costs

4. Basic AI Agent Concepts (1 week)

What is an AI agent?
Prompt engineering basics
Managing conversation context
Handling user input/output
